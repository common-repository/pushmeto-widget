== Description ==

This is a sidebar widget for the Pushme.to service. If you have a Pushme.to account and would like to publish a Pushme.to widget on your Wordpress blog - this is the widget you need. If you don't have a pushme.to account, but do have an iPhone or iPod Touch,  click [here](http://pushme.to/app/ "iTunes Pushme.to App") to open iTunes and download the app.


== Installation ==

1. Download the zip file and extract the contents.
2. Make sure you're running WordPress version 2.3 or better. It won't work with older versions.
3. Upload the folder to your plugins directory (wp-contents/plugins/)
4. Activate the plugin through the 'Plugins' page in WP.
5. Add the widget to your widgetized sidebar (Appearance->Widgets)
6. See widget options to set your Pushme.to nickname.
